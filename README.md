# my-map-doc

A simple app to show the technology that can be used to create a map app from a static web page.

Technology Stack
----------------

### Base
* HTML
* CSS
* BootStrap - http://getbootstrap.com/
* jQuery
* Google Maps V3

### Helpers
* tkmap 4.0.0
* infobox
* tabletop

### Project Specific
* projects
* event

### Site Specific
* local.js

### Other
* JSON
